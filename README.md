# Kinopoisk-React

Домашний проект для отрбаотки навыков работы в REACT.
В данном приложении реализованны : 
- Получение данных с API кинопоиска
- Рендер данных 
- Поиск по названию
- Поиск по жанрам (фиксированные значения)
- Получение и поиск по всем доступным жанрам  
- Сортировка полученных фильмов по году
